Zaero DeathMatch Pack
====================================
7/31/99 - Team Evolve

This Deathmatch pack will work with
both the demo, and the full version
of ZAERO.

Unzip it into your \Quake2\Zaero 
directory if you have the full
version of ZAERO.

directory if you have the demo
version of ZAERO.
